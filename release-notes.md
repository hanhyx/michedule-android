## 변경사항

### 개선
- 채팅 이미지 로딩 시 보라색 플레이스홀더 대신 로딩 스피너 표시
- 채팅 이미지 클릭 시 풀스크린 뷰어 (다운로드 기능 포함)
- 채팅 메시지 꾹 누르기 액션 메뉴 개선 (복사/삭제/닫기 아이콘)
- 채팅 메시지 글꼴 크기 증가
- 채팅 통신 속도 개선 (Realtime + 경량 폴링)
- 메모 삭제 시 상대방 동기화 반영
- 위젯 컨텐츠 확대 (메모, 할일, 감정메모 표시)
- 기본 프로필 아이콘 통일 (사람 아이콘)
- 앱 아이콘 업데이트

### 버그 수정
- 위젯에서 상대 일정 대신 내 일정이 표시되는 문제 수정
- 앱 재설치 시 이전 로컬 데이터 잔존 방지 (auto backup 비활성화)
